The Severe Weather Wisconsin application was created in December, 2020, as the final project in a geospatial web and mobile programming course during my master's program at UW-Madison. I created this project with 2 classmates which was done using front-end and back-end coding. The PostgreSQL geodatabase that this application accesses data from was created and hosted through pgAdmin. Java servlets were coded and used to allow the application's interface to communicate with our geodatabase. HTML, CSS, and JavaScript were used to create our application's interface page. This project was a useful learning experience in understanding how multiple coding languages can be used together to create products.

I created this infographic in February of 2020, while taking a graphic design in cartography course during my master's program at UW-Madison. We were assigned to create an engaging narrative using supporting data and a bivariate map, which we also made. This assignment was more meaningful because of the topic I choose about veterans and homeless veterans. Having many veteran family and friends, I understand that there are too many veterans that need help and support in ways they currently aren't receiving. I created this infographic to tell a story about where we are failing homeless veterans in the United States and that only proper support and resources can change this.
